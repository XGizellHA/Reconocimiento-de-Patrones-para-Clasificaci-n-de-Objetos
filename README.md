### Reconocimiento-de-Patrones-para-Clasificaci-n-de-Objetos
Este proyecto fue desarrollado para la materia de Reconocimiento de Patrones de la carrera de Ingeniería en Computación.  El objetivo fue clasificar diferentes objetos a partir de imágenes segmentadas utilizando técnicas de agrupamiento y reconocimiento de patrones.
